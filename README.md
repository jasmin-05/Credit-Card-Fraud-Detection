# Credit Card Fraud Detection using Machine Learning

## Project Overview
This project detects fraudulent credit card transactions using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Model
- Logistic Regression

## Results
- Accuracy: 99.05%
- Fraud Detection Prediction
- Confusion Matrix Visualization
